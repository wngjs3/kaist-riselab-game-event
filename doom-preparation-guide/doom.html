DOOM Tournament Preparation Guide

Introduction

Hello! This is the DOOM preparation guide for the DOOM x Chess Tournament. This guide will walk you through installing and running the ViZDoom client, an AI research platform for DOOM that we'll be using for our tournament.

Important: Please follow this guide to install and test the client before participating in the tournament. If you encounter any issues, contact us immediately.

Installing ViZDoom Client

1

Clone the GitHub Repository

First, download or clone the following GitHub repository:

Visit GitHub Repository

There are two ways to download the repository:

  1. Click the "Code" button on the GitHub page and select "Download ZIP" to download as a ZIP file
  2. Or run the following command in your terminal to clone:
git clone https://github.com/wngjs3/ViZDoom_Client_Setup.git
2

Run the Installation Script

Navigate to the downloaded repository directory and run the installation script appropriate for your operating system.

For macOS/Linux users:

cd ViZDoom_Client_Setup chmod +x install.sh ./install.sh

For Windows or other OS users:

cd ViZDoom_Client_Setup python install.py

The installation script will automatically install all necessary dependencies and set up the ViZDoom client.

Note: For macOS users, the installation script will automatically install necessary packages (cmake, boost, sdl2, wget). You may need to enter your password during script execution.

Running the ViZDoom Client

1

Launch the Client

Once installation is complete, you can run the client with the following commands:

chmod +x run_client.sh ./run_client.sh

Test Single Player Mode

To verify that the client is working correctly, test the single player mode:

  1. Click the "Single Player" button from the client's main screen

If the game runs normally in single player mode, your installation has been successful.

Troubleshooting: If you encounter issues running the client, please contact juheon@kaist.ac.kr. Include any error messages and your operating system information to help resolve the issue quickly.

Game Controls

The basic controls used in DOOM are as follows:

Key Action
W, A, S, D Movement (forward, left, backward, right)
Mouse movement Look/aim
Left mouse button Attack
Spacebar Jump
ESC Menu or exit game

In the tournament, you will be playing in deathmatch mode. The goal is to defeat as many opponents as possible to achieve a high score.

Frequently Asked Questions

I'm experiencing package installation errors. What should I do?

Try manually installing the required packages with the following command:

pip install numpy opencv-python matplotlib vizdoom pillow requests

The client won't start. What should I check?

Check the following:

  1. Verify that Python 3.6 or higher is installed
  2. Ensure all necessary dependencies were properly installed
  3. For macOS, check that required packages were installed via brew (cmake, boost, sdl2, wget)

If the problem persists, contact juheon@kaist.ac.kr.

I need additional help with installation.

We will provide installation help sessions before the tournament. Alternatively, you can contact juheon@kaist.ac.kr anytime for assistance.

Tournament Preparation Tips

Here are some tips to perform well in the tournament:

  1. Practice enough in single player mode to become familiar with game mechanics and controls.
  2. Experiment with different weapons and understand their strengths and weaknesses.
  3. Explore maps and memorize item locations and strategic points.
  4. Practice aiming at moving targets.
  5. Focus on practicing the maps that will be used in the tournament (E1M1, E1M7, E2M2).

Note: All participants must install and test the client before the tournament. It's best to resolve any technical issues before the day of the tournament.
